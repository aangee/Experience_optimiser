## Interfaces d'apprentissage interactif (recherche 2026-03-15)

Contexte : recherche pour le Mode Comprendre du portfolio (Phase 3).

### Khan Academy
- [Design Guidelines for Teaching Talkthroughs](https://blog.khanacademy.org/our-design-guidelines-for-teaching-programming-talkthroughs/)
  → Principes pédagogiques du "talkthrough" : montrer en live, erreurs volontaires, rester court
- [Live Editor (GitHub)](https://github.com/Khan/live-editor)
  → Éditeur open source ProcessingJS + canvas en temps réel
- [Live Editor Demo](http://khan.github.io/live-editor/)

### Bret Victor
- [Learnable Programming](https://worrydream.com/LearnableProgramming/)
  → Référence théorique fondamentale : Show the data, Show the flow, lien bidirectionnel, paramètres labelisés
- [Choc — Learnable Programming in JS](https://www.newline.co/choc/)
  → Implémentation JS des idées de Victor : step-through, timeline d'exécution

### Scratch
- [Scratch GUI (open source)](https://github.com/scratchfoundation/scratch-gui)
  → Composants React des cartes tutorielles flottantes
- [Scratch Tutorial Cards (wiki)](https://en.scratch-wiki.info/wiki/Scratch_Cards)

### p5.js
- [Web Editor](https://editor.p5js.org/)
- [Web Editor GitHub](https://github.com/processing/p5.js-web-editor)
- [Examples (beginner → avancé)](https://p5js.org/examples/)

### GDevelop
- [GDevelop Tutorials Repository](https://github.com/GDevelopApp/GDevelop-tutorials)
  → Tutoriels définis en JSON, moteur générique qui les consomme — pattern intéressant
- [GDevelop Academy](https://gdevelop.io/academy)

### Observable
- [Introduction to Notebooks](https://observablehq.com/@observablehq/introduction-to-notebooks)
  → Cellules réactives entrelacées (code + markdown + output)
- [A Better Way to Code — Mike Bostock](https://medium.com/@mbostock/a-better-way-to-code-2b1d2876a3a0)
- [Quarto + Observable JS](https://quarto.org/docs/interactive/ojs/)

### CodePen
- [Education Features](https://codepen.io/features/education)
  → Professor Mode, Collab Mode, Presentation Mode

### Articles & Analyses
- [Maggie Appleton — Making Programming Visual, Spatial, and Learnable](https://maggieappleton.com/learnable-programming)
